## 1. Overview

### 1.1 Purpose

그룹PT 센터 "근력학교 이화"의 수강생과 트레이너를 위한 통합 관리 웹앱을 구축한다. 수강생은 수업일정 확인, 대체수강/홀딩/시간변경 신청, 챌린지, 프로그램 열람, 훈련일지 확인 등을 할 수 있고, 트레이너는 수강생 관리, 수업 운영, 공지 작성 등을 할 수 있다.

### 1.2 Background

- 그룹PT 센터에서 수강생 관리를 효율적으로 하기 위한 전용 도구 필요
- 수강생 50~200명 규모의 중규모 PT 센터
- 현재 수작업/카카오톡 등으로 관리하던 것을 체계화
- 모바일에서 주로 사용할 반응형 웹앱

- Design: `docs/02-design/features/strength-school-ewha.design.md` (예정)

## 2. Scope

### 2.1 In Scope (MVP)

- [ ] 사용자 인증 (이름+전화번호 뒷자리 기반 초기 계정, 비밀번호 변경)
- [ ] 수업현황 (수강권 시작/종료일, 수강 진행률, 출석 달력, 통계)
- [ ] 대체수강 신청 및 관리
- [ ] 홀딩 신청 및 관리
- [ ] 정규반 시간 변경 신청 (월 1회, 자동승인, 만석 대기 예약)
- [ ] 챌린지 (출석 스트릭, 배지, 동기부여)
- [ ] 프로그램 열람 (트레이너 설정 훈련 프로그램, 읽기 전용)
- [ ] 훈련일지 (트레이너 작성, 수강생 열람)
- [ ] 공지사항 (트레이너 작성, 수강생 열람)
- [ ] 휴일 관리 (공휴일 기본 포함, 관리자 설정, 종료일 자동 연장)
- [ ] 프로필 관리 (사진 업로드, 닉네임, 아이디 변경, 회원탈퇴, 훈련일지 다운로드)
- [ ] 알림 기능 (수업/공지/대체수강/홀딩/시스템 알림, 읽음 처리)

### 7.2 Key Architectural Decisions

| 결정 항목 | 옵션 | 선택 | 근거 |

| API Client | fetch / axios / react-query | **TanStack Query + fetch** | 캐싱, 자동 갱신, Supabase와 호환 |
| Form Handling | react-hook-form / formik | **react-hook-form** | 성능 우수, 번들 사이즈 작음 |
| Styling | Tailwind / CSS Modules | **Tailwind CSS + shadcn/ui** | 빠른 개발, 반응형 우수, 일관된 디자인 |
| 날짜 처리 | date-fns / dayjs / Luxon | **date-fns** | 트리셰이킹 지원, 경량, 한국어 지원 |
| Backend | BaaS / Custom Server | **Supabase** | 무료 플랜, Auth+DB+Storage 통합, RLS 보안 |
| 배포 | Vercel / AWS | **Vercel** | Next.js 최적화, 무료 플랜, 간편 배포 |

| 8교시 | 20:00~21:10 | O | O | O | O | O |

- O = 수업 있음, - = 수업 없음
- 3~5교시는 운영하지 않음
- 정원: 교시당 최대 **8인**
- 토요일/일요일 수업 없음

### 8.2 여석 관리

- 실시간 여석 숫자 표시 (정원 8 - 등록 인원)
- 만석(여석 0)인 경우 대체수강 신청 불가
- 수업이 없는 시간대는 비활성화 처리

### 8.3 관리자 시간표 뷰 (정규반 시간표)

트레이너 대시보드에 표시되는 정규반 등록 수강생 시간표.

| 항목 | 설명 |
|------|------|
| **레이아웃** | HTML 테이블: 행=교시(숫자만: 1,2,6,7,8), 열=요일(월~금) |
| **교시 라벨** | 숫자만 표기 (1, 2, 6, 7, 8), 요일 헤더 글자 크게 (0.6875rem→0.75rem 데스크톱) |
| **셀 내용** | 등록 수강생 이름 2열×4행 그리드 (grid-template-columns: repeat(2, 1fr)) |
| **3카테고리 색상** | 일반=연보라색(rgba(179,157,219,0.25)/#7E57C2), 삐약이=다크골드(rgba(234,179,56,0.2)/#B8860B), 입학생=회색(rgba(158,158,158,0.12)/#888) |
| **배치 순서** | 일반(기본) → 삐약이(order:1) → 입학생(order:2, 하단 우측) |
| **여석 표시** | 별도 테이블: 만석=전체 셀 붉은 배경+"만석" 텍스트, 여석=숫자만, 수업 없는 날=빈칸(비워둠) |
| **표시 대상** | 정규 등록 수강생만 (대체수강 인원 제외) |
| **홀딩 표시** | 빨간색 글씨 + 사유 표시 (예: 장미라(미라) - 출산) |
| **수업 없는 날** | "—" 표시 + 비활성 스타일 |
| **수강생 검색** | 이름/별명 실시간 검색, 매칭 시 pulse+glow 하이라이트 애니메이션 |
| **모바일** | 가로 스크롤 (min-width: 680px) |
| **데스크톱** | full-width (사이드바 제외 전체 너비) |

- 1·2교시: 월·금만 수업 (화·수·목 비활성)
- 6교시: 월·수·금만 수업 (화·목 비활성)
- 7·8교시: 월~금 전일 수업

### 8.4 수강생 상세 모달

정규반 시간표·대시보드에서 수강생 이름 클릭 시 표시되는 상세 정보 팝업.

| 항목 | 설명 |
|------|------|
| **헤더** | 이름 (별명) + 삐약이만 🐥 이모티콘 (나머지 비움) + 닫기 버튼 |
| **기본 정보** | 요일/교시 (달력 아이콘, 예: "월금1" - 시간대 미표기), 수강권 (카드 아이콘, 기간 1줄 + 남은 횟수 2줄 분리 표시, 예: "26.02.01 ~ 26.02.28" / "12회 남음") |
| **금주 특이사항** | 대체수강(교환 아이콘, 요일+교시만 표기 "월 1교시 → 수 6교시"), 홀딩(X 아이콘, 붉은색 배경+아이콘, 요일+(사유)), 코칭 메모(✓ 아이콘, 앰버/골드 배경, 내용 줄바꿈) |

| **하단 버튼** | "상세보기 (출결 및 전체 기록)" - 진한 보라색 배경(var(--purple-600)), 흰색 텍스트 |

| **글자 크기** | 전체적으로 크게 (이름 1.25rem 본문폰트(CarmenSans), 정보 0.875rem, 라벨 0.8125rem, 아이콘 18px) |
| **블럭 간격** | 모든 블럭 동일 간격 (0.375rem) 유지 |

#### 입학생 분류

| 항목 | 설명 |
|------|------|
| **정의** | 등록 완료했으나 입학반 미수강으로 정규반 대기 중인 수강생 |
| **시각 표현** | 회색 배경 (rgba(158,158,158,0.12), color:#888) |


#### 헤더 메타 정보

| 필드 | 설명 | 예시 |
|------|------|------|
| 훈련일 | 수업 날짜 | 2025년 10월 22일 |
| 주차 | 프로그램 주차 | 2주차 |
| 일차 | 주 내 훈련 일차 (Day 1/2/3) | 2일차 |
| 상단 메모 | 오늘의 포커스/테마 | "피봇 week, 케틀벨 스킬 연습" |

#### 운동 기록 패턴 (3가지 유형)

| 유형 | 표기법 | 설명 | 예시 |
|------|--------|------|------|
| 직선 세트 | NXM | 동일 중량 반복 | 12X3 (12회 3세트) |
| 피라미드/드롭 | N-N-N | 횟수/중량 점진 변화 | 11-9-7, 20K→25K→30K |
| 시간 기반 | Nmin, Ns | EMOM/타이머/홀드 | 12min, 45s, 1분에 7개씩 |

#### %RM 표기

수강생이 1RM 기반 중량 산출을 직접 기록하는 패턴 확인.
예: "20X70% = 64K", "5X3 (60%)"

#### 특이사항/코칭 큐

자세 교정 및 주의사항이 주로 기록되는 항목:
- 자세 큐: "날개뼈 모으기", "팔꿈치 내리기!", "허리 붙이기"
